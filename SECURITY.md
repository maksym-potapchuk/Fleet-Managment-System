# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email**: [maksym.potapchuk5@gmail.com](mailto:maksym.potapchuk5@gmail.com)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You can expect an initial response within **48 hours**. I will keep you updated on the progress toward a fix and public disclosure.

**Do not** open a public GitHub issue for security vulnerabilities.
